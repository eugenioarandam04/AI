# LAWGEN

LAWGEN is now prepared as a static shareable webpage focused on lease analysis, credit calculations, and default-interest calculations.

## Share It

Because the current version is static HTML, CSS, and JavaScript, you can host it anywhere that serves static files, including GitHub Pages, Netlify, Vercel static hosting, or any simple web host.

The main entry file is [index.html](/Users/eugenioarandamtz/Documents/CODEX/index.html).

## What Changed

- The assistant panel remains in the design, but it is intentionally marked out of service.
- No backend or API key is required for this version.
- The calculators and written explanations still work normally in the browser.

## Notes

- This version is easier to share with a single public link.
- Since there is no live backend, there is no real chatbot in this build.
